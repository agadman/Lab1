Tailwind

Om Tailwind

Tailwind CSS är ett modernt CSS-ramverk som bygger på så kallade utility-klasser. I stället för att skriva egen CSS för varje komponent, kombinerar man färdiga klasser direkt i HTML-koden för att snabbt skapa responsiva och konsekventa layouter. Läs mer på tailwindcss.com.

Utvärdering

Jag tycker att Tailwind är ett smidigt ramverk när man vill bygga snygga gränssnitt snabbt, särskilt eftersom man kan styla allt direkt i HTML utan att växla mellan flera filer. Samtidigt kan koden bli lite svårläst med alla klassnamn, särskilt i större projekt. Men för mindre sidor eller prototyper tycker jag det fungerar riktigt bra.
